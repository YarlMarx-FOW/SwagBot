Проект вдохновлён ботом "пидорбот" и был создан одним человеком с помощью народа из общаги, которые подкидывают идеи для пасхалок и комманд
Автор осуждает всё, что прописано в листе банвордов, этот лист создан для осуждения подобных высказываний.

Бот просматривает каждое сообщение и с шансом в X% отвечает на него, используя свагоконцентрированную версию слова из него.
Логи НЕ ВЕДУТСЯ, просматривайте код. Мне не нужны чьи-либо данные и т.п., бот не создан для спама.

Чуть больше слов накопилось со временем. Теперь это не просто скрипт с тремя пасхальными командами как у того же пидорбота.
Свагабот это уже отдельная личность, душа компании, сын маминой подруги и т.п.

Логика свагинации меняется со временем общения в чате, "пасхалки" (кодовое название всех команд) постоянно добавляются, но уклон так же уходит именно в сторону лора чата с присущими ему мемами, пикчами, аудио и т.д.
Своим косноязычием я пытаюсь донести для случайного читателя то, что это именно проект для своих, с приколами и функциями для своих
